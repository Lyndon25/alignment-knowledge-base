## 一、论文结构与论证路线

### 1.1 核心创新：准表征概念
**准表征 (quasi-representation)** = 剥离了"关于性"(aboutness/intentionality) 后剩余的结构性成分：
- 相似性 (similarity)
- 结构同构 (structural isomorphism)
- 推理通行证 (inference ticket)
- 但不要求语义指称关系（关于某个真实世界对象的"关于性"）

### 1.2 论证结构
1. **标准科学无奇迹论证 (NMA)**：科学预测的成功将是奇迹，除非科学理论（近似）为真。
2. **AI版NMA**：Rowbottom等人在2024年早已论证NMA可扩展至AI——AI系统的预测成功最好由它们包含（近似）真实的表征来解释。
3. **自我修正 (2026)**：但2024年的版本太强——AI系统真的"表征"世界吗（带关于性）？作者现在主张：将论证从"表征"弱化为"准表征"——AI的预测成功最好由它们包含的**准表征**来解释。
4. **为什么更可辩护**：准表征对关于性保持沉默，因此避免了AI系统是否具有意向性的难题。

### 1.3 GraphCast案例
Google DeepMind的天气预测模型GraphCast包含大气状态的**结构性对应**（准表征），但不需要声称模型"关于"大气——结构对应足够支撑预测成功和可解释性。

## 二、哲学定位：中间立场的形式化

### 2.1 兼容性
准表征论题**同时兼容**：
- **选择实在论 (selective realism)**：准表征对应真实世界的结构性特征。你可以是准表征的实在论者——相信模型内部的特定结构性对应反映了自然界的真实结构。
- **反实在论 / 工具论**：准表征只是纯粹的建模工具。你可以接受准表征在解释上的有用性而不对其做任何实在论承诺。

### 2.2 中间立场的两面性
这既是优点也是弱点：
- **优点**：为超越传统的发现vs构建、实在论vs工具论二元对立提供了精致的理论空间
- **弱点**："准表征的实在论者"这一立场足够明确吗？还是只是一个模糊的"两者之间"？
- 如何区分"准表征对应真实结构"和"准表征是纯工具"这两个不同的承诺水平？这个区分在实践中是重要的——它决定了我们对准表征在alignment等应用中应给予多少信任。

### 2.3 对Culcu诊断的回应
Culcu (001) 诊断ML文献中0%结构实在论。Rowbottom的准表征概念表明：ML文献中0%强实在论（完全表征）是有道理的——强实在论确实难以为AI辩护。但这不意味着0%的**任何形式**的实在论——准表征层次的实在论是完全可能的。

## 三、对SAE实践的含义

### 3.1 SAE特征作为准表征
- SAE特征与模型内部激活模式之间有**结构性对应**（重构损失低意味着对应好）
- 但SAE特征不必然满足**关于性**——特征"Nudix box motif"确实对应某种激活模式，但它是否"关于"Nudix box——是否具有语义指称——是一个额外问题
- 准表征论证允许我们使用SAE特征进行干预（因为它们有结构保真度），而不需要解决关于性问题

### 3.2 对alignment的实际启示
- alignment不需要预设SAE特征是完全真实的（强实在论）或完全任意的（激进工具论）
- **准表征的结构性对应足够支撑可靠的因果干预**
- 但这也意味着"价值对齐"中的"对齐"可能是**结构对应**而非**语义对应**——这是一个深层问题

### 3.3 实践准则
来自准表征视角的操作性原则：
1. 对SAE特征的干预有效性只需要准表征保真度（结构对应），不需要语义真实性
2. 但从准表征到"alignment成功"的跃迁需要额外论证——仅仅结构对应可能不够
3. 准表征论题暗示，当前alignment研究可以安全地采用一种**方法论的实在论**（act as if features are real for practical purposes），同时保持**形而上学的不可知论**
