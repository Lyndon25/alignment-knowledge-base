# 002 | CAA: 激活操控的实证典范

**Rimsky et al. (Anthropic) 2023 | arXiv:2312.06681**

---

## 一、核心论点

### 1.1 对比激活添加（CAA）的基本原理

CAA的核心思路极其简洁：
- 收集模型在一对对比提示（如 harmful vs. harmless）下的激活值
- 计算两者之间的差异向量
- 将该向量乘以缩放系数后添加到推理时的残差流中

这一方法的优雅性来源于其直接性：它不训练任何额外组件，不修改模型权重，仅在推理时施加一个轻量级的线性干预。

### 1.2 认识论简单性

CAA在认识论上的优势是它的透明性——操控的效果可以直接被归因到特定的方向向量。这种清晰的因果链条使得CAA成为机械可解释性研究的理想实验工具。

## 二、方法论

### 2.1 Steering Vector构建

```
v_steer = (1/|D_harmful|) * Σ a(x_harmful) - (1/|D_harmless|) * Σ a(x_harmless)
```

### 2.2 推理时干预

```
a'_l = a_l + c * v_steer
```
其中c是缩放系数，l是目标层索引。

### 2.3 关键实验参数

- 模型：Llama 2 7B, 13B, 70B
- 对比数据：数百个有害/无害提示对
- 主要指标：Attack Success Rate (ASR), perplexity变化

## 三、核心发现

1. 简单的线性激活加法可有效控制大模型行为，无需微调
2. 拒绝向量可显著降低或增强模型的安全性响应
3. 中间层（~0.3-0.7总层数）的steering效果最佳
4. CAA的效果随模型规模增大而有所减弱
5. 方法仅需少量对比样本（几十到几百个）

## 四、与本研究的相关性

### 4.1 干预主义的理想实验

CAA恰好符合Woodward干预主义因果理论的实验结构：
- **目标变量**：模型行为（拒绝 vs 合规）
- **干预**：向残差流添加方向向量
- **测量**：行为变化（ASR, perplexity）

这使得CAA成为分析'干预作为因果理解工具'的理想技术案例。

### 4.2 作为学科基线的价值

CAA的真正价值不仅在于它作为技术工具的成功，更在于它提供了足够清晰、可被系统超越的基线。后续工作几乎全部建立在对CAA的某种批评或改进之上：
- AlphaSteer（004）：改进安全-效用的权衡
- COAST（005）：形式化附带损伤问题
- FGAA（010）：扩展到SAE潜在空间
- Scaling Laws（011）：研究CAA的规模行为

这种'清晰基线→系统改进'的发展模式是科学进步的典型形态。

### 4.3 隐含的哲学预设

CAA依赖若干未明确陈述的哲学预设：
1. **线性可分解性**：行为的因果结构在表征空间中是线性可分解的
2. **概念统一性**：'有害'作为一个统一的概念存在
3. **干预独立性**：对refusal方向的操控不（显著）影响其他功能

后续工作逐步挑战了这些预设，使该领域从朴素实在论向更复杂的表征哲学演进。

## 五、开放问题

1. CAA提取的方向向量在多大程度上是'真实'的概念方向，而非只是任务artifact？
2. 为什么中间层的steering最有效——这是否指示了概念信息在transformer层级结构中的特定分布？
3. 线性假设在什么条件下失效？如果失效，是否意味着我们需要不同的操控范式？
4. CAA的跨模型迁移性：不同模型中的同一'概念方向'是否对应？
